# Crypto App

A React Native crypto app built with **Expo**, **Expo Router**, and **React Navigation**.

---

## Features

* Bottom tab navigation
* Vector icons & custom fonts
* Smooth animations with Reanimated
* Haptics feedback support
* Works on iOS, Android, and Web

---

## Installation

```bash
npm install
npm run start      # Start Metro server
npm run android    # Android device
npm run ios        # iOS simulator
npm run web        # Web browser
```

---

## Scripts

| Script          | Description           |
| --------------- | --------------------- |
| `start`         | Start dev server      |
| `android`       | Run on Android        |
| `ios`           | Run on iOS            |
| `web`           | Run in browser        |
| `reset-project` | Reset project         |
| `lint`          | Check for lint errors |

---

## Tech Stack

* React Native 0.81.5
* Expo SDK 54
* Expo Router & React Navigation
* TypeScript & ESLint
